## Keyboard Shortcuts
A reference for common VS Code shortcuts.

---

### Navigation

| Action | Windows / Linux | Mac |
|--------|----------------|-----|
| **Quick Open** (find any file) | `Ctrl+P` | `Cmd+P` |
| **Command Palette** | `Ctrl+Shift+P` | `Cmd+Shift+P` |
| **Go to Symbol in File** | `Ctrl+Shift+O` | `Cmd+Shift+O` |
| **Go to Line** | `Ctrl+G` | `Ctrl+G` |
| **Toggle Sidebar** | `Ctrl+B` | `Cmd+B` |
| **Switch between open editors** | `Ctrl+Tab` | `Ctrl+Tab` |
| **Go back / forward** | `Alt+Left` / `Alt+Right` | `Ctrl+-` / `Ctrl+Shift+-` |

---

### Editing

| Action | Windows / Linux | Mac |
|--------|----------------|-----|
| **Find in file** | `Ctrl+F` | `Cmd+F` |
| **Find & Replace in file** | `Ctrl+H` | `Cmd+H` |
| **Find across all files** | `Ctrl+Shift+F` | `Cmd+Shift+F` |
| **Format document** | `Shift+Alt+F` | `Shift+Option+F` |
| **Move line up / down** | `Alt+Up` / `Alt+Down` | `Option+Up` / `Option+Down` |
| **Duplicate line** | `Shift+Alt+Down` | `Shift+Option+Down` |
| **Delete line** | `Ctrl+Shift+K` | `Cmd+Shift+K` |
| **Toggle line comment** | `Ctrl+/` | `Cmd+/` |
| **Multi-cursor (click)** | `Alt+Click` | `Option+Click` |

### Building & Deploying Code

These commands are run from the integrated terminal (`` Ctrl+` ``).

| Task                        | Command                                     |
|-----------------------------|---------------------------------------------|
| **Build Robot Code**        | `Ctrl+F5` → **WPILib: Build Robot Code**    |
| **Deploy Robot Code**       | `F5` → **WPILib: Deploy Robot Code**        | 
| **Run a VS Code task**      | `Ctrl+Shift+P` → **Tasks: Run Task**        |
| **Run the last task again** | `Ctrl+Shift+P` → **Tasks: Rerun Last Task** |
| **View build errors**       | `Ctrl+Shift+M` (Problems panel)             |

> **Tip:** Use `Ctrl+Shift+P` → **Tasks: Run Task** to see all available tasks defined in `.vscode/tasks.json`.
