# What Are We Trying to Do?
The main role of robot programming is to make the robot 'act'. This involves taking an input and programming the robot to (generally) provide a desired output

## Types of Inputs
- User Input (TeleOp)
- Sequence of Actions (Autonomous)
- Sensors (Automated)

## Types of Outputs
- Motors
  - Linear
  - Rotational
- Sensors
  - LEDs
- Virtual
  - Logs
